xarray datasets#

Introduction#

In this notebook, we take code from simple test of get_landsat_scene and use it to call a new function: sat_lib.landsat_read.read_landsat_dataset. This is very similar to the get_landsat_scene function used in simple test of get_landsat_scene, with two improvements:

  1. It takes an optional list of bands in the form ['B01','B02',...] with the default being ['B04','B05,'B06']. The Fmask cloud mask is also returned.

  2. Instead of returning a dictionary it returns an xarray.Dataset which is a collection of xarrays sharing the same coordinates (in this case x,y in the map projection). Datasets have a variety of useful features, but the most important one for work in this class is the ability to write the Dataset to disk, via xarray.Dataset.to_netcdf – demonstrated below.

Below we repeat the plot from simple test of get_landsat_scene using the new function and saving all channels to disk in a file called week9_landsat.nc

from matplotlib import pyplot as plt
from sat_lib.landsat_read import get_landsat_dataset
from rasterio.windows import Window

function arguments#

You can pass any combination of valid landsat/sentinel bands in as a list in the form
[‘B01’,’B02’,…] – the default is for the ndvi bands ‘B04’,’B05’ plus the near-ir ‘B06’ with ‘Fmask’ always included.
